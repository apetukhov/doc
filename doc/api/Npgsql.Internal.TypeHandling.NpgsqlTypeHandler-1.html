    Class NpgsqlTypeHandler<TDefault>

    Base class for all type handlers, which read and write CLR types into their PostgreSQL binary representation. Unless your type is arbitrary-length, consider inheriting from NpgsqlSimpleTypeHandler<TDefault> instead.

    Type Parameters
    Name Description
    TDefault

    The default CLR type that this handler will read and write. For example, calling on a column with this handler will return a value with type TDefault. Type handlers can support additional types by implementing INpgsqlTypeHandler<T>.
